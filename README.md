## Object-head-Museum
一个专注于二次元虚拟头像分享的微信小程序社区。

> 基于 Vue3 + uni-app 开发，严格遵循微信小程序与法律法规合规规范，所有功能适配个人主体上架，适合前端/小程序开发求职展示。

---

### 一、项目定位

- 面向二次元、虚拟形象、覆面爱好者的公益社区
- 支持异头、OC、虚拟头像图片图文分享与互动
- 平台全程无盈利行为，确保个人主体类小程序合规上线

---

### 二、合规规范（核心风险规避）

- **无私密互动**：仅公开发布/评论/点赞/收藏，绝无私信/一对一功能
- **无经营行为**：无支付、无打赏、无虚拟币、无会员
- **无位置功能**：不获取用户地理信息，不做同城/附近匹配
- **内容规范严控**：仅允许二次元类头像内容，禁止真人/低俗/擦边
- **内容安全审核**：对接微信官方内容安全API自动审核图片/文字
- **举报屏蔽合规**：支持用户举报、账号屏蔽、内容删除等必备合规功能
- **用户隐私保护**：仅用微信授权，不收手机号、身份证等敏感信息
- **无诱导行为**：绝无强制分享/关注等违规诱导

---

### 三、技术栈

- **前端框架**：Vue3 + `<script setup>`
- **跨端**：uni-app（适配微信小程序）
- **状态**：Pinia + 本地持久化
- **UI组件**：uview-plus / uv-ui
- **网络请求**：uni.request 二次封装、统一错误处理
- **安全校验**：微信内容安全API（文本、图片双重校验）
- **其他**：ESLint、分包加载、图片压缩、懒加载、节流防抖

---

### 四、核心功能

#### 1. 首页
- 瀑布流布局，贴合二次元风格
- 分类/排序与大数据量下拉刷新、上拉加载

#### 2. 发布页
- 支持多图上传、图片压缩预览、分类标签
- 发布时自动内容安全校验

#### 3. 详情页
- 图片放大、下载指引，支持公开评论、点赞、收藏
- 举报不良内容，后台可人工审核

#### 4. 个人中心
- 我的发布、收藏、喜欢
- 昵称/头像同步微信、不收敏感数据
- 提供账号注销、隐私政策、用户协议等合规页面

#### 5. 简易后台
- 人工二次审核、违规内容下架/删除
- 评论/用户管理、禁言、屏蔽词库管理

### 六、运行/上线说明

1. 用 HBuilderX 打开，`npm install` 安装依赖
2. 在 `manifest.json` 中填入个人小程序 AppID
3. 微信开发者工具本地调试、接口对接及合规测试
4. 上线前补充隐私/协议页面，上传后等官方审核
